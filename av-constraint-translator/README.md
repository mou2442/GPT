# 视听约束转译器 v0.1

## 定位

一个把**静帧图片**整理成：
- 结构包
- 运动表
- 风格约束

并输出为：
- JSON 中间层配置
- 模型投喂文本

的纯前端工作台。

它不是普通提示词拼接器，而是一个用于**视频生成前处理**的中间层工具。

---

## 当前入口

- 页面文件：`index.html`
- 在线地址：`https://mou2442.github.io/GPT/av-constraint-translator/`

---

## v0.1 已实现

- 静帧图片上传与预览
- 结构包编辑
- 运动表编辑
- 风格约束编辑
- JSON 输出预览
- 约束文本导出
- 经验包本地保存（localStorage）
- 皮影示例一键填入

---

## v0.1 边界

当前版本**不做**：
- 自动提轮廓
- 自动分层
- 自动关键点识别
- 视频 / 多帧轨迹计算
- 云端数据库
- 第三方模型直连

也就是说，当前版本是：
**人工主导的约束编排器**。

## 说明

如果要在 GitHub 网页端继续加文件，不必先找“新建文件夹”按钮。
直接新建例如：

