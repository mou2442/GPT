## GitHub 界面里“文件夹”为什么不好找

GitHub 网页端通常**不是先建空文件夹**，而是：
- 新建文件时，直接在文件名里写路径
- 例如：`av-constraint-translator/README.md`
- 提交后，这个目录就会自动出现

也就是说，GitHub 里的文件夹通常是**被文件带出来的**，不是先空着等你建。
